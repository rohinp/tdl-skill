# Resource-Grounded Deep Dives

Resource-grounded deep dives explore a supplied resource when it is necessary for mastery or requested by curiosity.

## Use Cases

- Walk through a book chapter.
- Inspect internal company documentation.
- Build exercises from slides.
- Compare a learner's answer to a specific tutorial.
- Clarify a prerequisite introduced by a resource.

## Rules

- Keep the supplied resource as the source of truth.
- Track the resource ID and section.
- Separate resource-backed statements from external context.
- Return to the parent topic after the branch objective is met.

## Output

End with:

- resource sections covered;
- concepts learned;
- open questions;
- relation to parent topic;
- restored checkpoint and next challenge.
