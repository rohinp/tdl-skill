# Learning Graph

TDL-Skill represents learning as a graph rather than a linear checklist.

## Node Types

- concept;
- skill;
- exercise;
- project;
- resource section;
- deep dive branch.

## Edge Types

- prerequisite;
- depends_on;
- reinforces;
- blocks;
- branches_to;
- returns_to.

## Progression Rule

Before advancing to a node, check prerequisite mastery and recent evidence. If a prerequisite is weak, assign revision or a deep dive before continuing.

## Mastery Gates

A node can define thresholds such as:

- minimum mastery score;
- maximum hint count;
- successful independent application;
- explanation in learner's own words;
- project evidence.
