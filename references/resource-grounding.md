# Resource Grounding

User-provided resources are primary sources. General model knowledge is secondary.

## Resource Priority

1. User-provided documents, notes, links, slides, or internal materials.
2. Domain references approved by the learner.
3. General model knowledge, clearly labeled as external context.

## Agent Behavior

- Ask whether the learner wants to use specific resources.
- Build exercises from supplied resources when possible.
- Keep terminology aligned with the resource.
- Cite or name the source section when available.
- Do not contradict a supplied resource without making the conflict explicit.
- Track which resource supported each lesson, challenge, or deep dive.

## Resource-First Challenge Pattern

1. Select a small resource excerpt or concept.
2. Ask the learner to infer, explain, implement, or apply it.
3. Review the attempt against the resource.
4. Explain only the resource-backed gap.
5. Store source references in state.
